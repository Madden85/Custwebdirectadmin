CUSTOMER WEBSITE 2 - TOYYIBPAY V1

Files to deploy:
- index.html
- index2.html
- app.js
- app2.js
- success.html
- all image files (.jpg, .png)

Important:
1. Website Control Apps Script must already be V12 ToyyibPay Success Return.
2. app2.js API_URL is set to the new Website Control Web App URL.
3. PROMO_CONTROL needs Normal Price for packages that can be sold.
4. Promo Active = ON only for actual promo.
5. STOCK_CONTROL / live stock controls whether customer can buy.
6. After customer pays, ToyyibPay returns to success.html?orderId=... and success.html opens Telegram admin with paid order message.
